Title: Стили для ссылки, с переходом в новое окно
Content: Для того чтобы открыть ссылочку в новом окне мы используем атрибут **target** со значением **_blank**.

```
<a href="#" target="_blank">open link in new window</a>
```
<!-- pagebreak -->
Что если, для таких ссылок, задавать свои правила CSS? Я думаю стоит подумать об этом. Если сайт посещаемый, можно научить подльзователя различать такие ссылки. В примере ниже на такой ссылке выводится тултип. Либо можно менее очевидно, окрашивать в другой цвет либо использовать другое подчеркивание.
